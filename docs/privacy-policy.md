# Kozukase 隱私權政策

**最後更新日期：2026-05-21**
**生效日期：2026-05-21**

Kozukase（以下簡稱「本平台」、「我們」）依《個人資料保護法》（以下簡稱「個資法」）及相關法令處理您的個人資料。本政策說明我們如何蒐集、處理、利用、保護及刪除您的個人資料。當您註冊或使用本平台時，即表示您已閱讀、理解並同意本政策全部內容。

> **重點摘要**
> - 我們只蒐集**提供服務所必需**的個人資料，**不販售**給第三方。
> - 您可隨時查詢、更正、刪除自己的個人資料。
> - 密碼由認證服務商加密儲存，本平台**無法**取得明文。
> - 賣家的社群帳號採**一次性私訊驗證碼**確認持有人，本平台不會取得您的社群帳號密碼或長期登入授權。
> - 站內私訊**僅對話雙方可見**，平台不主動審查。

## 第一條　個人資料之蒐集

### 1.1 蒐集目的

本平台蒐集您的個人資料，是為了：提供與管理代購媒合服務（帳號管理、商品與賣家搜尋、評價與互動功能）、保護消費者並處理交易與爭議、維護網站運作與資通安全、進行統計分析以改善服務，以及在您同意的範圍內進行行銷與發送通知。若未來新增金流相關服務，我們會另行更新本政策說明。

### 1.2 蒐集的資料類別

**1.2.1 註冊與帳號管理（所有使用者）**

| 資料類別 | 是否必要 | 蒐集來源 |
|---------|---------|---------|
| 電子郵件 | 必要 | 您主動提供 / Google 登入 |
| 密碼（雜湊） | 視註冊方式 | 您主動提供；由認證服務商雜湊儲存，本平台無法取得明文 |
| 使用者代號 | 必要 | 您主動提供（全平台唯一） |
| 顯示名稱 | 必要 | 您主動提供 / Google 帳號資料 |
| 頭像 | 選填 | 您主動上傳 / Google 帳號資料 |
| 註冊 / 最後登入時間 | 自動產生 | 系統 |
| 第三方登入識別碼 | 視登入方式 | Google |

**1.2.2 賣家資料（僅限申請成為賣家者）**

- **公開資料**：賣家名稱、自我介紹、頭貼、代購地區（用於賣家頁面顯示與買家搜尋）。
- **聯絡資料**：手機號碼（僅供身分驗證與必要聯繫，**不公開**）。
- **社群帳號驗證**（選填）：賣家可驗證自己的 Instagram、Threads 帳號，以在賣家頁面顯示已驗證標記。驗證時，您需從自己的社群帳號私訊一組驗證碼給本平台官方帳號，我們據此確認帳號持有人，並記錄您的社群帳號名稱（Instagram 另記錄發送者的帳號識別碼）。本平台**不會取得您的社群帳號密碼或長期登入授權**，您可隨時於賣家後台取消驗證。

**1.2.3 行為資料（所有使用者）**

- **內容資料**：您上架的商品、代購、連線公告；發布的評價、評分、回覆；提交的檢舉。
- **互動資料**：許願、追蹤、收藏、通知讀取狀態。
- **站內私訊**：對話文字、附圖與引用的商品 / 連線資訊。**僅對話雙方可見**，平台不主動審查。
- **瀏覽紀錄**：刊登瀏覽、賣家主頁瀏覽、社群連結點擊等事件，用於提供賣家儀表板統計與平台改善，**不會與第三方廣告聯播網交換**。
- **活躍資料**：登入紀錄、最後上線時間（內部使用，不於前台公開）。

**1.2.4 技術資料（自動蒐集）**

包含 IP 位址、瀏覽器與作業系統資訊、使用語言與時區、Cookie（含登入狀態）、操作紀錄，以及服務發生錯誤時的請求路徑、瀏覽器版本與錯誤訊息（傳送至第三方錯誤監控服務）。用途為身分驗證、安全防護、防詐欺、在地化與故障排除。

> 本平台採用**自建分析機制**記錄上述瀏覽與點擊事件，僅用於賣家儀表板與平台統計，**目前未使用** Google Analytics、Meta Pixel 等第三方行為分析工具；若未來新增，將更新本政策並通知。

### 1.3 蒐集方式

來自三種管道：您主動提供（註冊、編輯、上架、評價、社群帳號驗證時）；經您授權的第三方登入（Google）；以及您訪問時瀏覽器自動傳送的 HTTP 標頭與 cookie。

### 1.4 不蒐集的資料

本平台**不主動蒐集**身分證 / 護照號碼、信用卡或銀行帳戶、健康與醫療紀錄、性傾向 / 宗教 / 政治傾向、犯罪前科、生物特徵等敏感資料。若您於評論、留言、檢舉中**自願揭露**上述資訊，由您自行承擔風險。

## 第二條　個人資料之利用

### 2.1 利用範圍

我們在下列範圍內利用您的個人資料：

- **核心服務**：帳號驗證登入、個人化頁面、商品與賣家搜尋、評價 / 許願 / 追蹤 / 收藏 / 通知、檢舉處理與內容審核、賣家社群帳號驗證。
- **帳號管理**：安全防護與異常登入偵測、客服與糾紛協助、違規調查。
- **服務改善**：去識別化後的統計分析、系統錯誤排除、介面與演算法調校。
- **法令義務**：依法配合司法 / 行政機關調取資料、主管機關稽查。
- **公告與通知**：系統維護、條款修訂、安全警示、重要服務變更。

### 2.2 我們不會做的事

我們**不會**販售您的個資、未經同意用於行銷、用於本政策未載明之目的，也不會將手機號碼等敏感資料公開揭露於介面（僅供內部驗證與必要聯繫）。

### 2.3 利用之地區與期間

您的資料可能於本平台所委託之雲端服務供應商（提供資料庫與身分驗證、網站託管、檔案儲存、錯誤監控等後端作業）所在地處理與儲存。我們已確認上述供應商符合相應的資料保護標準。跨境傳輸之細節見第七條。

我們僅於達成蒐集目的所必要的期間內保有您的資料。原則上，帳號、站內私訊與您發布的內容保留至您刪除帳號為止（私訊的對話對方仍可能持有副本）；但依稅法、商業法等法令或爭議處理需要保留的紀錄（如交易相關紀錄、檢舉紀錄），會於帳號刪除後以匿名化方式保留至多 7 年。系統、錯誤監控、瀏覽與安全稽核等技術性紀錄，僅於維運與安全所需的必要期間內保存後刪除；社群帳號驗證碼則於驗證完成或失效後即刪除。

## 第三條　個人資料之公開與分享

### 3.1 哪些資料會公開、哪些不會

**公開於平台**（含未登入訪客可見）：使用者代號、顯示名稱、頭像；成為賣家後另含賣家名稱、自我介紹、代購地區、您已驗證的社群帳號；以及您發布的代購 / 連線內容、評價評分等互動資料。

**不對外公開**（僅本人及必要時的平台管理可見）：電子郵件、密碼、手機號碼、各類登入與帳號識別資訊、IP 與瀏覽器資訊、社群帳號驗證碼、站內私訊、個人通知與收藏 / 許願 / 追蹤清單、瀏覽與最後上線紀錄，以及系統與稽核紀錄。

### 3.2 分享給第三方的情形

我們僅於下列情形將個資分享給第三方：

**(1) 服務供應商（必要委外處理）** — 各供應商均依其隱私政策處理，且僅限為本平台提供服務之目的。

我們委託雲端服務供應商處理資料庫與身分驗證、網站託管、檔案儲存、錯誤監控等後端作業；其中密碼僅以加密形式儲存，錯誤監控資料僅含必要的請求路徑與瀏覽器版本。

此外，下列為您會直接接觸或主動授權的整合服務：

| 服務 | 處理範圍 | 目的 |
|------|---------|------|
| Google | 您主動授權之資訊 | 登入認證 |
| Meta（IG / Threads） | 本平台官方帳號收到的私訊驗證碼 | 核對驗證碼以確認社群帳號持有人 |

**(2) 法令要求**：司法機關搜索票 / 傳票 / 命令、主管機關查調公文、為防止人身重大危害或緊急犯罪偵防、為保護本平台或他人合法權益。

**(3) 業務轉讓**：若發生合併、收購、業務或資產移轉，您的個資可能隨業務一併移轉。我們將於移轉前以顯著方式通知，您可選擇於移轉前刪除帳號。

**(4) 取得您明示同意**：除上述情形外，分享前必先取得您的明示同意。

### 3.3 我們不會分享的事

除法令要求或您明示同意外，我們**不會**將您的資料用於第三方行銷、出售予資料經紀商、提供予廣告聯播網做行為分析，或進行跨平台合併追蹤。

## 第四條　個人資料之保護

### 4.1 安全措施

**技術上**：加密傳輸與儲存（密碼僅以加密形式儲存，無法還原為明文）、分層權限控管、登入安全防護、上傳檔案檢查、錯誤監控資料最小化。**組織上**：僅限必要人員存取個資、相關人員定期接受個資保護訓練、管理員操作均留存稽核紀錄、依角色劃分權限。

### 4.2 資料外洩通報

若發生個資外洩，我們將立即啟動應變程序阻斷來源並評估範圍，並依個資法第 12 條於查明後儘速以電子郵件、平台公告等適當方式通知受影響使用者（說明外洩的資料類別、原因、已採取的補救措施及您可採取的自我保護措施）；如法令另有通報主管機關之要求，我們將依規定辦理，事後並分析原因、強化防護。

### 4.3 安全限制

沒有任何系統能完全免於攻擊。請您妥善保管登入憑證、使用強密碼且不重複使用、留意釣魚網站與惡意連結。因您自身保管不慎所致的損害，本平台不負賠償責任。

## 第六條　使用者權利

依個資法第 3 條，您對自己的個人資料享有下列權利：

- **查詢與製給複本**：可於帳號設定頁查詢，或來信要求提供個資副本（電子檔，最長 30 日處理）。
- **補充更正**：可於帳號設定頁更正，無法自行更正者來信申請。
- **停止蒐集、處理或利用**：可要求停止；但若因此無法提供服務，本平台得停止對您的服務（如刪除帳號）。
- **刪除**：可於帳號設定頁刪除帳號（流程見下）。
- **拒絕行銷**：可隨時透過信件連結或站內設定拒絕行銷訊息；但交易與帳號相關的必要通知（帳號異常、條款修訂、安全警示）不在此限。
- **申訴**：可向本平台、向直轄市 / 縣（市）政府，或向個人資料保護委員會等主管機關申訴。

**行使方式**：部分權利可於帳號設定頁直接操作，其餘請寄信至服務信箱並載明使用者代號、電子郵件、申請事項與聯絡方式。我們將於 **30 日內**回覆；如需延長將事前通知。

**刪除帳號的處理**：點選「刪除帳號」並確認後，系統會將個人識別資料（電子郵件、手機、顯示名稱、使用者代號、頭像）匿名化；對外公開的內容（評價、上架歷史）依需要保留為「已刪除使用者」；私人資料（許願、收藏、追蹤、通知、最後上線時間）刪除；站內私訊中的識別欄位改為「已刪除使用者」，但訊息本文仍保留於對話對方的收件夾，本平台無法代為刪除對方端副本；社群帳號驗證資料完全刪除。因法令或爭議處理需保留的資料（交易、檢舉、稽核紀錄）於保留期間屆滿後自動刪除。

**拒絕之情形**：若申請有妨害國家或他人重大利益、妨害公務機關執行職務、涉及法定保留資料，或顯不合理 / 濫用權利等情形，我們得拒絕並說明理由。
