# 賣家上架紅線指引

> **用途**：這份是給賣家上架時參考的「紅線清單」，幫你避免誤觸法規、被下架或受罰。
> 正式的義務規定請見[使用者條款](./terms-of-service.md) 第四條 4.1.7；本指引是把它講得更白話、更具體，方便對照。

---

## 一、不得上架的商品

下列商品**一律不得在平台上架**：

- **藥品、醫療器材**（不論是否經主管機關核准）
- 法令禁止販售，或須**特許資格**才能販售的商品
- 毒品、管制藥品、武器刀械
- 仿冒品、侵害智慧財產權的商品
- 保育類動植物及其製品（象牙、龜殼等）
- 含肉、動物產品等入境管制品

---

## 二、受限商品：可賣，但要小心

**食品、保健食品、化妝品**可以上架，但你必須：

1. 自行確認**販售資格、輸入合法性、標示完整**（成分、賞味期限等）。
2. **不得宣稱醫療效能，也不得誇大、不實宣傳。**

> 提醒：化妝品 / 食品「帶回來自用」沒事，但「轉賣」涉及販售資格與輸入規定，違規最高可罰數十萬至數百萬元，責任由賣家自負。

---

## 三、禁止出現的宣稱字眼（紅線）

依食藥署廣告規範，下列宣稱**禁止出現在商品標題、描述或圖片中**：

### 🔴 化妝品
| 類型 | 例字 |
|------|------|
| 治療 / 醫療 | 治療、治癒、根治、消炎、殺菌、抗菌、抗敏、解毒、生髮 |
| 改善疾病 | 改善 / 預防（濕疹、異位性皮膚炎、青春痘…）、除疤、淡疤 |
| 醫療等級 | 醫療級、醫美級、藥用、處方、皮膚科認證 |
| 細胞 / 換膚 | 活化細胞、促進細胞再生、修復受損肌膚、換膚、溶脂、瘦臉 |
| 美白針類 | 美白針、淡斑針，或任何「打 / 吃就變白」的說法 |

### 🔴 食品 / 保健品
| 類型 | 例字 |
|------|------|
| 療效 | 治療、預防 / 改善疾病、根治、療效 |
| 特定疾病 | 降血壓 / 血糖 / 血脂、抗癌、防癌、消腫瘤、護肝、壯陽 |
| 誇大功效 | 排毒、燃脂、瘦身、塑身、增高、豐胸（宣稱有效時） |

### 🔴 通用誇大 / 不實（任何商品都別寫）
100% 有效、永久、立即見效、最有效、第一、唯一、無副作用、保證療效、官方授權 / 授權代理（非真實時）

---

## 四、可以怎麼寫（安全範圍）

✅ 描述「使用感受」與「事實」沒問題：

- 保濕、滋潤、清潔、修飾、調理膚觸
- 日本直送、代購採買、實物拍攝照片、如實標示成分

**原則：講「使用感受」和「事實」可以，宣稱「醫療效果」不行。**

---

## 五、違規後果

- 平台得依[使用者條款](./terms-of-service.md)逕行**下架商品、移除違規內容、停權帳號**。
- 上架商品的合法性、廣告內容及由此所生的一切責任，**由賣家自行承擔**。
