# 買家流程說明

## 頁面與路由總覽

| 路由 | 頁面 | 需登入 |
|---|---|---|
| `/` | 首頁 | 否 |
| `/search` | 商品搜尋 / 瀏覽 | 否 |
| `/products/[id]` | 商品詳細頁 | 否 |
| `/listings/[id]` | 上架詳細頁 | 否 |
| `/sellers/[id]` | 賣家主頁 | 否 |
| `/wishes` | 許願榜（公開排名） | 否 |
| `/connections` | 連線代購瀏覽 | 否 |
| `/profile` | 個人頁面（收藏、許願、追蹤、評價） | 是 |
| `/notifications` | 通知 | 是 |
| `/account` | 帳號設定 | 是 |
| `/settings` | 成為賣家 | 是 |

---

## 各頁面說明

### 首頁 `/`
- Hero 搜尋欄，引導使用者輸入關鍵字
- 8 個商品分類快捷鍵（時尚穿搭、美妝保養、食品零食、3C 電器、生活雜貨、公仔玩具、限定聯名、其他）
- 點分類 → 跳轉至 `/search?category=xxx`

### 商品搜尋頁 `/search`
- 支援 query string：`?q=`（關鍵字）、`?category=`（分類）、`?sort=latest|price_asc`
- 桌機左側篩選欄 / 手機底部 sheet
- 無限捲動商品格

### 商品詳細頁 `/products/[id]`
- 商品圖片 gallery、名稱、品牌、分類
- **所有賣家上架比較**：依價格 / 運送天數 / 規格比較各家
- 許願按鈕（顯示許願數）
- 收藏商品按鈕
- 點某筆上架 → 進入上架詳細頁

### 上架詳細頁 `/listings/[id]`
- 上架圖片 gallery、價格、出貨天數、規格、賣家備註
- **複製詢問文字**：預格式化的訊息，買家複製後去 IG / Threads 私訊賣家
- **查看原始貼文**：跳轉至賣家的 IG / Threads 貼文
- 收藏此上架按鈕
- 檢舉功能
- 賣家資訊卡（頭像、名稱、評分、認證徽章），可點擊進入賣家主頁

### 賣家主頁 `/sellers/[id]`
- 賣家頭像、名稱、社群認證徽章、平均評分、加入時間
- 服務地區標籤、IG / Threads 連結
- 追蹤 / 取消追蹤按鈕
- 三個 tab：
  - **上架商品**：賣家目前有效上架的格
  - **評價**：其他買家的評價清單 + 留評表單（需登入）
  - **連線代購**：賣家目前在某國的即時代購資訊

### 許願榜 `/wishes`
- 顯示全站最多人許願的商品排名（公開）
- 無限捲動

### 連線代購 `/connections`
- 顯示目前在海外可即時代購的賣家
- 每張卡顯示地點、日期區間、描述、賣家資訊
- 點連線 → 賣家主頁

## 需登入的頁面

### 個人頁 `/profile`
四個 tab：

| Tab | 內容 |
|---|---|
| 收藏 | 收藏的商品 + 收藏的上架 |
| 許願 | 自己的許願清單（上限 20 個） |
| 追蹤 | 追蹤中的賣家格 |
| 我的評價 | 自己留過的評價（附賣家名稱、星級、留言） |

### 通知 `/notifications`
| 通知類型 | 觸發時機 |
|---|---|
| `new_listing_for_wish` | 許願的商品有賣家新上架 |
| `followed_seller_new_listing` | 追蹤的賣家新增上架 |
| `wish_product_removed` | 許願的商品被管理員移除 |
| `bookmarked_product_removed` | 收藏的商品被管理員移除 |
| `review_liked` | 自己的評價被按讚 |

### 帳號設定 `/account`
- 修改顯示名稱
- 更換頭貼

---

## 核心互動行為

| 行為 | 需登入 | 說明 |
|---|---|---|
| 搜尋 / 瀏覽 | 否 | 完全公開 |
| 許願商品 | 是 | 上限 20 個；許願數影響許願榜排名 |
| 收藏商品 | 是 | 儲存至個人頁「收藏」tab |
| 收藏上架 | 是 | 儲存至個人頁「收藏」tab |
| 追蹤賣家 | 是 | 有新上架時收到通知 |
| 留評價 | 是 | 在賣家主頁「評價」tab 留評（星級必填，留言選填） |
| 詢問購買 | 否 | 複製預格式化詢問文字，自行至 IG / Threads 聯絡賣家 |

> 注意：平台**沒有直接下單功能**，交易在 IG / Threads 上完成。

---

## 完整動線

```
首頁
 └─ 搜尋 / 點分類
      └─ 搜尋頁
           └─ 點商品
                └─ 商品詳細頁（許願、收藏）
                     └─ 點上架
                          └─ 上架詳細頁（複製詢問文字、收藏上架）
                               └─ 點賣家
                                    └─ 賣家主頁（追蹤、留評）

連線代購頁
 └─ 點連線 → 賣家主頁

許願榜
 └─ 點商品 → 商品詳細頁
```
