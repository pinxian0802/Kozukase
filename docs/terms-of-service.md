## 第四條　成為賣家

### 4.1 申請條件

1. 使用者須先完成買家註冊。
2. 須提供以下資訊：
   - **賣家名稱**（必填，1-50 字元）
   - **手機號碼**（必填，請提供有效且本人持有之號碼）
   - **代購地區**（必填，至少一項）
3. 選填：頭像、自我介紹、Instagram / Threads 帳號授權連結

### 4.2 賣家義務

賣家除遵守一般使用者義務外，另負下列義務：

1. **資訊真實**：所有上架商品、代購資訊、價格、出貨時間、規格說明須真實正確，不得虛偽不實。
2. **誠信履約**：對買家詢問須誠實回應，承諾事項應確實履行。
3. **合規代購**：所代購之商品須符合來源國與輸入國之法令，包括但不限於：
   - 不得代購屬於管制、違禁、違法之商品（如毒品、武器、保育類動植物、未經核准之藥品 / 醫療器材、仿冒品等）
   - 須遵守關稅、消費稅、進口許可等規定
   - 須遵守智慧財產權法令
4. **個人資料保護**：賣家於代購過程中取得買家之個人資料（如姓名、地址、電話），僅得用於完成代購交易，不得作為其他用途，並應妥善保管、不得外洩。
5. **稅務責任**：賣家因代購所得之收入，依法應自行向稅務機關申報並繳納相關稅負，本平台不協助稅務申報亦不代為扣繳。
6. **客服與爭議處理**：買賣爭議由賣家自行與買家協調解決，本平台不介入。

### 4.3 上架限制

1. 每位賣家最多同時上架 **25 個代購商品**。
2. 每位賣家最多同時發布 **5 個連線公告**。
3. 上架代購必填欄位：商品、貼文連結、出貨天數、價格（或「私訊報價」）。
4. 連線公告必填欄位：地區、開始日期、結束日期。
5. 圖片上傳：每筆代購 / 連線最多 **5 張**，每張圖片不得超過 **5MB**，僅接受 JPEG、PNG、WebP 格式。

### 4.4 賣家停權

若賣家有下列情事之一，本平台得逕行停權：

1. 違反本條款或社群守則
2. 提供虛偽不實資訊
3. 涉嫌詐欺、未交付商品、惡意下架
4. 多次被檢舉且查證屬實
5. 違反代購相關法令
6. 其他經本平台認定有礙平台秩序或損害他人權益之行為

停權後果：

- 帳號標記為 `is_suspended`
- 所有上架代購自動下架
- 所有連線公告自動結束
- 搜尋結果中隱藏該賣家
- 賣家收到通知

4. 您可隨時於賣家後台中斷連結，中斷後本平台立即刪除相關 token 並清除快取資訊。
